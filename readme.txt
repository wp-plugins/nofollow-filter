Nofollow filter is the easiest way to add "nofollow" to links on your content.
== Description ==

Nofollow Filter is a Wordpress plugin that automatically adds “nofollow” attribute to links you choose.

== Installation ==

1. Upload 'nofollow-filter' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add sites on "Settings" -> "Nofollow filter"
